== Description ==

It's a platform for new application regarding vacant role and user simply submit after filling out the necessary information along (doc, docx, pdf) resume file to increase their chances for bright future.

== Installation ==

This section describes how to install the plugin and get it working.

1. Fetch, install and activate the plugin.
2. Use the [resume_upload_form] shortcode for form
3. Use the [resume_upload_form_list] shortcode for resume list. Click on resume files to review them.
4. Find uploaded file at this path `/wp-content/uploads/year/month` directory.
5. Click on checkbox under the email field if user also want to get its response as acknowledged.
